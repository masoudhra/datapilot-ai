# Agent Spec - DataPilot AI

## Overview
DataPilot AI is an AI agent designed to help users prepare for Data Analyst interviews. It runs locally as a prototype and is built using Google ADK 2.0.

## Example Use Cases
### Use Case 1: SQL Question Generation
- **Input**: User asks for a question or says "Let's start the interview" or similar.
- **Output**: The agent generates a realistic SQL interview question (including table schemas and a clear problem description).

### Use Case 2: Answer Evaluation & Feedback
- **Input**: User submits a SQL query as their answer.
- **Output**: The agent evaluates the query correctness, provides constructive interview-style feedback, and suggests a short learning recommendation.

## Tools Required
- `validate_sql_syntax`: A simple Python tool to perform a basic syntax check/mock evaluation on the user's SQL query.

## Constraints & Safety Rules
- The agent must ONLY focus on Data Analyst SQL interviews.
- Never write credentials or hardcode secrets.
- Use a single `LlmAgent`.
- Keep everything local. No cloud deployment, no database, no frontend, no auth.

## Success Criteria
- Successfully generates SQL questions.
- Successfully parses and evaluates SQL answers.
- Provides actionable feedback and learning recommendations.
